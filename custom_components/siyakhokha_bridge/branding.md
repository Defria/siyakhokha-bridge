# Branding assets

All artwork in this repository is original to this project. No City of Ekurhuleni or Siyakhokha portal assets are bundled.

Files:

- `logo_banner.png` — horizontal banner used in the inner README header.
- `favicon.ico` — multi-size icon (16/32/48 px) generated from the same source SVG.
- Source SVG lives at `branding/logo.svg` in the repo root, rendered to PNG/ICO via cairosvg + Pillow.

The square logo used in the top-level `README.md` is `../../logo_square.png` in the repo root.

Concept: an itemised receipt with a "PAID" stamp — original artwork by the project author. Palette: dark navy background (#0B1A24 / #102434), cream paper (#FBF7EE), teal accent (#0E9F8A / #1FE0C2), red stamp (#D94A3D), amber accent (#F4B23C).
